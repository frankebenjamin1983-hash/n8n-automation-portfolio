# KI-basierte Lead-Qualifizierung

## Problem
Ein Berater erhält 20+ Kontaktanfragen pro Woche – viele sind nicht relevant, echte Leads bleiben liegen.

## Lösung
KI-gestützter n8n-Workflow:
1. Eingehende Anfragen (E-Mail, Kontaktformular, LinkedIn) sammeln
2. Per KI bewerten: Budget, Dringlichkeit, Passgenauigkeit → Score (A/B/C)
3. A-Leads: sofortige persönliche Benachrichtigung + E-Mail-Entwurf
4. B-Leads: automatische Antwort + Qualifizierungs-Fragebogen
5. C-Leads: Newsletter-Abo mit Drip-Kampagne

## Ergebnis
- 🎯 Fokus auf die 20% wichtigsten Leads
- ⏱️ 2–3 Std./Woche weniger E-Mail-Sortierung
- 📈 Höhere Abschlussrate durch schnellere Reaktionszeit

## Technologie
- n8n (Webhook + E-Mail-Trigger)
- Ollama/OpenAI (Lead-Bewertung)
- Google Sheets (Lead-Datenbank)
- E-Mail (automatische Antwort-Serien)
