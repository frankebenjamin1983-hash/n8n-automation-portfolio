# n8n Automation Portfolio – Dokumentation

## Struktur

```
/
├── README.md                     # Hauptseite (Positionierung, Leistungen, Kontakt)
├── workflows/                    # Beispiel-Workflows mit Dokumentation
│   ├── angebotserstellung/       # Automatische Angebotserstellung aus E-Mail
│   ├── terminmanagement/         # Multi-Channel-Terminbuchung mit KI
│   ├── kunden-crm/               # CRM-light mit Google Sheets + Mail
│   ├── rechnungs-automation/     # Rechnungserstellung & Mahnwesen
│   └── lead-qualifizierung/      # KI-basierte Lead-Bewertung
├── docs/
│   ├── leistungen.md             # Detaillierte Leistungsbeschreibung
│   ├── preise.md                 # Preisübersicht
│   └── angebot-vorlage.md        # Angebotsvorlage für Kunden
└── templates/                    # Vorlagen & Referenzen
```

## Für Kunden

Dieses Portfolio richtet sich an:
- **Handwerksbetriebe** (1–20 MA), die Zeit & Kosten sparen wollen
- **Lokale Dienstleister** mit repetitiven Büro-Abläufen
- **Coaches & Solopreneure** mit Digitalisierungsbedarf

**Schmerzpunkte**, die wir lösen:
- Angebote manuell per E-Mail → automatisch aus Vorlage
- Termin-Chaos über WhatsApp → zentrales System mit KI
- Kundendaten in Excel → strukturiert im CRM-light
- Rechnungs-Follow-up vergessen → automatisierte Mahnstufen
